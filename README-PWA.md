# PWA Setup - Tilføj til hjemmeskærm

Appen er nu konfigureret som en Progressive Web App (PWA) og kan tilføjes til hjemmeskærm på både Android og iOS.

Appen bruger `public/icon.svg` direkte som ikon - ingen generering af PNG filer nødvendig.

## Sådan tilføjer du appen til hjemmeskærm

### Android (Chrome)
1. Åbn appen i Chrome
2. Klik på menuen (3 prikker) → "Tilføj til hjemmeskærm"
3. Bekræft navnet og klik "Tilføj"
4. Appen vil nu åbne i fuldskærm uden browser UI

### iOS (Safari)
1. Åbn appen i Safari
2. Klik på del-knappen (firkant med pil op)
3. Vælg "Tilføj til hjemmeskærm"
4. Bekræft navnet og klik "Tilføj"
5. Appen vil nu åbne i fuldskærm uden browser UI

## Funktioner

- ✅ Standalone mode (fuldskærm uden browser UI)
- ✅ App ikoner
- ✅ Theme color
- ✅ iOS specifikke indstillinger
- ✅ Portrait orientation
